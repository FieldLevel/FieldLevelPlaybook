# Alert

Use the Alerts component to display critical page-load messages, such as warnings, errors, or informational updates. For task confirmations or user actions, use the Feedback component instead. Alerts provide clear, accessible context for important messaging.

#### Critical

Used to communicate urgent issues that require immediate user attention and potentially corrective action.

#### Warning

Used to alert users to potential issues or situations requiring caution but not immediate action.

#### Highlight

Used to draw attention to important but non-critical information or updates.
