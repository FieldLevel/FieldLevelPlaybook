# Typography

A more limited set of typography variation is available with more distinct variations available via components with clear usage guidelines.

## Text Size

Used only for body text variation. Otherwise use typography components.

|                |                                   |
| -------------- | --------------------------------- |
| `text-body-sm` | <TextStyle name="text-body-sm" /> |
| `text-body`    | <TextStyle name="text-body" />    |
| `text-body-lg` | <TextStyle name="text-body-lg" /> |

## Font Weight

Bold is available to add some weight variation.

|               |                                  |
| ------------- | -------------------------------- |
| `font-normal` | <TextStyle name="font-normal" /> |
| `font-bold`   | <TextStyle name="font-bold" />   |

## Letter Spacing

For variation with letter spacing.

|                  |                                     |
| ---------------- | ----------------------------------- |
| `tracking-tight` | <TextStyle name="tracking-tight" /> |
| `tracking-loose` | <TextStyle name="tracking-loose" /> |
