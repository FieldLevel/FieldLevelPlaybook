# Text

Playbook exposes text-related CSS utilities through Tailwind. Use these utilities to customize font size, weight, and letter-spacing. See Titles and text to find all typographic components.

## Body

Use Body for all general-purpose application text needs. Body comes in base, small, and large variants. All body styles share a common line-height for easy stacking. Use large and small variants sparingly.

|                |                                   |
| -------------- | --------------------------------- |
| `text-body-sm` | <TextStyle name="text-body-sm" /> |
| `text-body`    | <TextStyle name="text-body" />    |
| `text-body-lg` | <TextStyle name="text-body-lg" /> |

## Weight

Text in Playbook comes in two weights: normal and bold. Use bold when you need to create contrast between text of the same size.

|               |                                  |
| ------------- | -------------------------------- |
| `font-normal` | <TextStyle name="font-normal" /> |
| `font-bold`   | <TextStyle name="font-bold" />   |

## Letter Spacing

Use letter-spacing to kern custom type. Do not kern text in Titles and Text components.

|                  |                                     |
| ---------------- | ----------------------------------- |
| `tracking-tight` | <TextStyle name="tracking-tight" /> |
| `tracking-loose` | <TextStyle name="tracking-loose" /> |
