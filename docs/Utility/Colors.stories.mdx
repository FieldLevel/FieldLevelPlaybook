# Colors

Playbook provides functional naming conventions for color to communicate usage clearly. We use names like critical, highlight, and warning, not red, green, or purple, so that the use of each color is apparent.

## Text Color

|                  |                                        |
| ---------------- | -------------------------------------- |
| `text-base`      | <TextColor name="text-base" />         |
| `text-muted`     | <TextColor name="text-muted" />        |
| `text-disabled`  | <TextColor name="text-disabled" />     |
| `text-critical`  | <TextColor name="text-critical" />     |
| `text-warning`   | <TextColor name="text-warning" />      |
| `text-highlight` | <TextColor name="text-highlight" />    |
| `text-success`   | <TextColor name="text-success" />      |
| `text-on-dark`   | <TextColor name="text-on-dark" dark /> |
|                  |                                        |
